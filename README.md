# Fairness First Design Thinking for Software Architecture

The replication package is organized as follows



## Repository Structure
This is the root directory of the repository. The directory is structured as follows:

	rep-pkg
	 .
     |
     |--- [edu-guide](./edu-guide/) contains the following files                             
            |
            |--- [lecture-slides.pdf](./edu-guide/lecture-slides.pdf)		  		Overview of the core concepts and lecture material relevant for the assignment.                                
            |
            |--- [usecase-guide.pdf](./edu-guide/usecase-guide.pdf)  				A worked-out example to help students structuring and describing a use case scenario.                               
            |
            |--- [VPandP_Reference.pdf](./edu-guide/VPandP_Reference.pdf)			Quick reference card summarizing concepts related to architecture viewpoint and architecture perspective.                             
            |
            |--- [perspective-template.pdf](./edu-guide/perspective-template.pdf)	Template for documenting architecture perspective for the assignment.                            
            |
            |--- [solution-template.pdf](./edu-guide/solution-template.pptx)        Template for assignment solution.                 
            |
            |--- [grading-rubric.pdf](./edu-guide/grading-rubric.pdf) 				Grading rubric for the assignment                            
            |
            |--- [assignment-guide.md](./edu-guide/assignment-guide.md)				Assignment description and reference material.
	 |
     |--- [raw](./raw/) contains the anonymized assignments from students
	 |
     |--- [extracted-data.xlsx](extracted-data.xlsx) contains the data extracted from the raw data
